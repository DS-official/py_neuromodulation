run_Bay_Opt(space, rounds=10, base_estimator='GP', acq_func='EI', acq_optimizer='sampling', initial_point_generator='lhs')[source]¶

Run skopt bayesian optimization skopt.Optimizer: https://scikit-optimize.github.io/stable/modules/generated/skopt.Optimizer.html#skopt.Optimizer

example: https://scikit-optimize.github.io/stable/auto_examples/ask-and-tell.html#sphx-glr-auto-examples-ask-and-tell-py

Special attention needs to be made with the run_CV output, some metrics are minimized (MAE), some are maximized (r^2)

Parameters
spaceskopt hyperparameter space definition
roundspython:int, optional

optimizing rounds, by default 10

base_estimatorpython:str, optional

surrogate model, used as optimization function instead of cross validation, by default “GP”

acq_funcpython:str, optional

function to minimize over the posterior distribution, by default “EI”

acq_optimizerpython:str, optional

method to minimize the acquisition function, by default “sampling”

initial_point_generatorpython:str, optional

sets a initial point generator, by default “lhs”

Returns
skopt result
